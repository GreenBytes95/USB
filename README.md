USB
=========================

Библиотека для эмуляции устройств ввода, с использованием управляющих устройств.

**Описание:**
Библиотека предоставляет доступ к устройству.

**Поддерживаеммые микроконтроллеры:**
* Arduino Leonardo (32U4 , 16U4)
